Acerca de este proyecto Acerca de este proyecto My First Ionic App

Proyecto desarrollado con fines académicos en la materia de desarrollo de aplicaciones móviles, de la carrera de Ingeniería informática del TecNM Campus Región Sierra.


La finalidad de este proyecto es mostrar las funcionalidades básicas de Ionic y Angular, para el desarrollo de aplicaciones móviles.

En este proyecto se ha utilizado el framework Ionic, que es un framework de código abierto que permite desarrollar aplicaciones móviles híbridas, es decir, aplicaciones que se ejecutan en un navegador web, pero que se comportan como aplicaciones nativas.
